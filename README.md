# Homework 1: Coding Assistant Onboarding

This repository contains my work for Homework 1, which focused on setting up a coding assistant workflow, using the assistant to create a file in a project folder, making my own manual edits, and tracking the process with Git and GitHub. For this assignment, I used Claude to generate an initial draft and then revised the final version myself to make it more personal, specific, and aligned with my own interests in Healthcare and Clinical AI.

## Repository Contents

- `assistant_draft.md` — the initial draft generated with the help of Claude
- `final_version.md` — the manually revised version with my own edits
- `README.md` — a summary of the workflow, Git steps, commit history, and video link

## Git Workflow

```bash
git init
git remote add origin https://github.com/TianjinDuan01/hw1-tianjinduan
git add README.md assistant_draft.md final_version.md
git commit -m "Initial repo setup"

# After generating the draft with Claude:
git add assistant_draft.md
git commit -m "Add assistant draft"

# After adding the prompt and reflection notes:
git add assistant_draft.md
git commit -m "Add reflection on AI assistance"

# After manually editing final_version.md:
git add final_version.md
git commit -m "Revise final version manually"

# After updating the README:
git add README.md
git commit -m "Update README"

# After adding the walkthrough video link:
git add README.md
git commit -m "Add video link"

git push origin main
```

## Commit Summary

- `Initial repo setup` — created the repository and the three required files
- `Add assistant draft` — added the Claude-generated draft to `assistant_draft.md`
- `Add reflection on AI assistance` — added the required prompt and reflection notes to `assistant_draft.md`
- `Revise final version manually` — edited `final_version.md` myself to make it more personal and specific
- `Update README` — added the repository description, Git steps, and commit summary
- `Add video link` — added the walkthrough video link to the README

## Reflection

This exercise helped me practice using a coding assistant as part of a real file-based workflow rather than just as a chatbot. One thing I found useful was how quickly the assistant could turn my resume and instructions into a structured first draft. At the same time, I found that the strongest version still required my own editing to make the writing more concise, more personal, and better aligned with what I actually wanted to say.
